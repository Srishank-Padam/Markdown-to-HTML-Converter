This is a simple Python command-line tool that converts a `.md` (Markdown) file into a `.html` (HTML) file.  
It supports:

- Headings (`#`, `##`, `###`)
- Bold (`**bold**`)
- Italic (`*italic*`)
- Inline code (`` `code` ``)
- Links (`[text](url)`)
- Unordered lists (`- item`)
